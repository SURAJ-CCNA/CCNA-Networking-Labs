# CCNA Static Routing Lab

## Overview
This project demonstrates the implementation of static routing in Cisco Packet Tracer using a multi-router topology. The lab verifies end-to-end communication between two LANs through manually configured static routes.

## Topology
- 5 Cisco ISR4331 Routers
- 2 Cisco 2960 Switches
- Multiple End Devices (PCs/Laptops)
- Point-to-Point Router Links (/30 Subnets)

## Technologies Used
- Cisco Packet Tracer
- Cisco IOS CLI
- IPv4 Addressing
- Static Routing
- Subnetting

## Skills Demonstrated
- IPv4 Address Planning
- /30 Point-to-Point Network Design
- Router Interface Configuration
- Static Route Configuration
- Next-Hop IP Configuration
- CLI Troubleshooting
- End-to-End Connectivity Testing

## Network Verification
- Successful Ping between LANs
- Successful Traceroute
- Interface Verification (`show ip interface brief`)
- Routing Table Verification (`show ip route`)

## Learning Outcome
This lab helped me understand how routers forward packets using manually configured static routes, how next-hop IP addresses work, and how to troubleshoot routing issues in Cisco networks.
